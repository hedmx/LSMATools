## 1. 项目概述

**LSMATools_WIFS**（Water Image + In-phase Full-Sequence）是一套腰椎矢状位 MRI 自动分割与几何量化工具，专为 T2 Dixon Water/In-phase 双序列设计。

### 核心能力

| 能力 | 说明 |
|------|------|
| 自动切片优选 | 从 3D NIfTI 中选取最优矢状位中间切片（V14_2 三步合并+相对评分） |
| 椎管自动追踪 | SpinalCanalProcessor 完整分割椎管，输出皮质线1 |
| Mode4 IN图膜态分割 | 加载同患者 IN 序列，沿皮质线2-2 扫描终板汇合点，**disc 模式**矩阵扫描终板 + 扇形扫描前缘 |
| 椎体几何量化 | 输出四交点（AP_top/AP_bot/BP_top/BP_bot）及形态角度 |
| 四种格式输出 | NIfTI 掩模（多标签）+ Fiji ROI ZIP + CSV 几何表 + PNG 可视化 |
| 批量处理 | 全量模式 / 快速轻量模式，自动匹配 T2 Dixon W 序列 |

### 适配设备

- 西门子 Dixon：`_W`（水相）、`_IN` / `_INPHASE`（同相位）
- 联影 Dixon：`_W`（水相）、`_IP`（in-phase，等同 IN）

## 3. 系统架构与数据流

```
NIfTI (W图 3D)
      │
      ▼
[preprocessing]
  series_utils   → 序列类型识别
  slice_selector → 切片优选 → best_slice_idx, green_mask, canal_seed
      │
      ▼
[segmentation]
  SpinalCanalProcessor → traced(椎管掩模), v9_data{all_rows, smooth_cols}
  皮质线1 (c1_rows, c1_cols)
  皮质线延伸 → 皮质线2-2 (c2_rows_mode4, c2_cols_mode4)
      │
      ▼
[preprocessing]
  image_loader → IN图 (in_img_2d, pixel_spacing)
      │
      ▼
[detection – Mode4 流水线]
  Step1: signal_ref       → low_mean, high_mean
  Step2: junction_detector→ junction_pts, anchor_pts_list
  Step2b: repair_junction → 去多/补少/末尾实扫补全
  Step3: disc_centers     → disc_centers, vert_centers
  Step3.5: verify_last    → 末尾两椎体汇合点校正
  Step4: fan_scanner/disc → scan_results[{sup_pts,inf_pts,ant_pts,ant_dirs,fan_params/None}]
  Step4c: _verify_ant_pts → 前缘候选点二次校验（夹角<65°且头颅侧上25°区间删除）
  Step5: anterior_edge    → cluster_results[{sup,inf,ant}]
      │
      ▼
[chain]
  build_vertebra_chain → vertebra_chain, ant_line
      │
      ▼
[output]
  export_masks    → {stem}_seg.nii.gz + {stem}_roi.zip
  export_csv      → {stem}_geom.csv
  export_log      → {stem}_log.txt
  visualize_wifs  → {stem}_vis.png
```

## 5. 模块详述

### 5.1 preprocessing – 预处理

#### series_utils.py

| 函数 | 说明 |
|------|------|
| `_get_series_type(desc)` | 从 series_description 识别序列类型，返回 `'W'/'IN'/'F'/'OPP'/None`。支持后缀：`W/WFI/WATER → W`；`IN/INPHASE/IN_PHASE/IP → IN`（IP 为联影 in-phase 命名） |
| `_is_dixon_sequence(desc)` | 判断是否为 Dixon 序列（描述中含 DIXON 关键词，不区分大小写） |

#### image_loader.py

`find_in_image(nifti_path, slice_idx)` — 在 W 图同目录或父目录中搜索 IN 序列，返回 `(in_img_2d, in_meta, in_nii_path)`。

匹配策略（按优先级）：
1. `_get_series_type(desc) == 'IN'`
2. 宽松匹配：`series_description` 末尾分段含 `'in'` 或 `'ip'`（兼容联影 `_IP` 命名）

#### slice_selector.py

`select_best_slice(data, pixel_spacing)` — V14_2 三步掩模合并 + 相对评分。

**候选切片**：以 3D 数据中间切片为基准，前后各2张，共5张候选。

**三步掩模合并算法**：

| 步骤 | 操作 |
|------|------|
| Step1 | 上区域（0%~65%行）Otsu 分割，取最大连通域作为"核心绿色掩模"（锁定椎管身份） |
| Step2 | 下区域（30%~95%行）各连通域与核心掩模像素重叠判断，有重叠则合并（同一椎管延伸） |
| Step3 | 若合并后底行 < 60% 图像高度，向下搜索 20mm 窗口找候选区域，**四重形态校验**全部通过才桥接 |

**Step3 桥接形态校验**（全部通过才合并）：

| 校验项 | 条件 | 说明 |
|--------|------|------|
| ① 宽度比 | `0.5 ≤ w_region / w_green ≤ 1.5` | 候选区域宽度须与核心掩模相当，防止非椎管结构混入 |
| ② 左边界自洽 | `max_adjacent_jump ≤ max(3px, 0.20 × med_w)` | 候选区域自身左边界相邻行跳变不超过阈值，不依赖 green 尾部比对（因桥接点截断面双方均非真实趋势） |
| ③ 宽度一致性 CV | `CV ≤ 0.50` | 候选区域内各行宽度变异系数不能过大，保证形态稳定 |
| ④ 细长比 | `height / width ≥ 1.5` | 椎管应为长条状，避免块状非椎管区域 |

**评分权重**（宽度异常切片强制0分）：

```
total_score = 0.45 × 面积评分 + 0.45 × 底部位置评分 + 0.10 × 空洞惩罚评分
```

**宽度异常检测**：逐行扫描绿色掩模，≥5行宽度超过 30mm → 强制0分。

**全零兜底**：选中间候选切片，对掩模做宽度修正（按背侧锚点向腹侧保留中位数宽度×1.2）。

**返回值**：`(best_idx, best_green_mask, best_area, merged_regions, best_canal_seed, csf_hints)`

### 5.3 detection – 特征检测

#### _scan_utils.py

| 函数 | 说明 |
|------|------|
| `_project_to_c2(r0, c0, c2_rows, c2_cols)` | 将点投影到皮质线2，返回 `(r_c2, c_c2, n_row, n_col)`，其中法线朝腹侧（`n_col < 0`） |
| `_scan_normal_descent(...)` | 沿方向扫描找第一个信号低谷（终板/椎间盘用），Step2确认：横向两两组合 `(c-1,c)` 或 `(c,c+1)` **任一均值** < 阈值 |
| `_scan_normal_descent_diag(...)` | 终板专用低谷扫描（对角确认版），Step2确认：`(r,c)+(r-1,c+1)` 或 `(r,c)+(r+1,c-1)` **任一均值** < 阈值（`/` 方向）。用于最后椎体后缘夹角 < 65° 时 |
| `_scan_normal_descent_ant(...)` | 前缘专用低谷扫描，Step2确认：纵向两两组合 `(r-1,r)` 或 `(r,r+1)` **任一均值** < 阈值 |
| `_scan_normal_descent_ant_diag(...)` | 前缘专用低谷扫描（单像素确认版），Step2确认：**当前像素单点信号** < 阈值。用于最后椎体后缘夹角 < 65° 时（S1 前缘信号确认方式） |

> **forward_check 已完全删除**：四函数不再包含 `forward_check` 参数和相关逻辑。

**低谷判断逻辑**（四函数共享 Step1；Step2 各有差异）：
```
Step1 - 下降沿触发（四函数一致）：
  cur < ref × (1 - drop_ratio)
  ref = 前2点滑动均值（正常模式）
       或 ref_locked（锁定模式，见下）

  【ref 锁定机制】：下降沿触发但 Step2 确认失败时，
  锁定当前 ref（ref_locked = ref），防止 ref 随低值漂移导致后续阈值下沉。
  解锁条件：cur ≥ ref_locked × (1 - drop_ratio / 2)（信号回升时自动解锁）

Step2 - 区域信号双条件确认（满足其一即通过）：
  【_scan_normal_descent】：(c-1,c) 或 (c,c+1) 任一均值 < 阈值
  【_scan_normal_descent_diag】：(r,c)+(r-1,c+1) 或 (r,c)+(r+1,c-1) 任一均值 < 阈值
  【_scan_normal_descent_ant】：(r-1,r) 或 (r,r+1) 任一均值 < 阈值
  【_scan_normal_descent_ant_diag】：单像素值 < 阈值（S1 前缘专用）

  阈值条件（双条件任一满足）：
    region_mean < in_high_mean × 0.55
    OR
    region_mean < in_low_mean × low_ratio   # low_ratio=1.3

drop_ratio 按场景分别取值：
  终板扫描（上/下终板）：drop_ratio = 0.25
  前缘扫描（标准椎体）：anterior_drop_ratio = 0.35
  前缘扫描（S1 最后椎体）：ant_drop = 0.20（信号对比度低，单独放宽）
```

#### signal_ref.py — Step1

`compute_signal_references(in_img_2d, c2_rows, c2_cols, pixel_spacing, offset_mm=20.0)`

沿皮质线2**前70%点**（头侧段，排除末尾30%骶椎高信号区）每个点法线方向偏移 20mm 采样 IN 图信号，统计：
- `low_mean`：低于中位数的采样均值（椎间盘/脑脊液信号参考）
- `high_mean`：全图非零像素 90 百分位（骨/软组织高信号参考）

#### junction_detector.py — Step2 / Step2b

##### Step2 主扫描：scan_endplate_junction_points

```
沿皮质线2-2 从顶向下滑动窗口（2mm×2mm）
每个c2点沿法线腹侧偏移offset，计算窗口内均值
offset 分段线性渐变：
  行程 0%~60%：offset = 1mm → 3mm（线性渐变）
  行程 60%~100%：offset 固定 3mm
滑动均值平滑（smooth_win=3）
低于 low_mean×0.8 的区间取中点 → 终板汇合点
静默距离：8mm（初始扫描忽略 c2 起点后 8mm 内的检测）
```

##### Step2b 修补：repair_junction_pts

三步校验：

**A. 去多（假阳性删除）**：
- 遍历相邻三点，若两侧间距之和 < `1.5 × median_gap`
- 按两侧间距长短决定删哪个：`gap_prev ≤ 0.75×median` 时删当前点；`gap_next ≤ 0.75×median` 时删下一点
- 循环直到无变化

**B. 补少（漏检插值）**：
- 若某相邻间距 > `1.8 × median_gap`，在皮质线2弧长等分位置插入补全点

**C. 末尾实扫补全**：
- 计算末尾剩余路径 `tail_remaining`，最多补 `min(4, floor(tail_remaining/gap))` 个点
- 实扫策略：沿 c2 路径状态机滑动，每个 c2 点逐档测试 offset（-2mm → 16mm，步进 1mm，共19档），找到低信号即停止
- 负值 offset（-2mm~0mm）向背侧探测，扩展骶椎低信号探测范围
- 静默距离：从末尾最后一个已知汇合点起 20mm 内不触发新检测
- 候选段封口：连续高信号中断 → 取候选段中点作为汇合点
- 兜底：无 IN 图时按路径等分预测插入

#### disc_centers.py — Step3 / Step3.5

##### Step3：compute_disc_and_vertebra_centers

每个终板汇合点沿皮质线2法线延伸（渐变 15mm→20mm）→ 椎间盘中心点  
过近点合并（< 5mm 距离合并）  
相邻椎间盘中心点中点 → 椎体中心点

##### Step3.5：verify_last_junction_point（末尾两椎体独立校验）

触发条件：**无条件执行**末尾两个椎体的独立校验（不再有夹角触发阈值）

夹角计算：
- `angle_last`：`junction_pts[-2]` → `junction_pts[-1]` 连线与水平线夹角
- `angle_2nd`：`junction_pts[-3]` → `junction_pts[-2]` 连线与水平线夹角

校验流程：
1. 从 `last_junction_pt` 投影到皮质线2，取投影点法线方向（确保朝腹侧：`n_col < 0`）
2. 根据夹角动态计算偏移量（`angle_deg=None` 时默认 15mm）：
   ```
   offset_mm = 8 + (clamp(angle_deg, 40°, 80°) - 40) / 40 × 12
   ```
   → 40° 对应 8mm，80° 对应 20mm
3. 从皮质线2投影点沿法线延伸 `offset_mm` → 候选椎间盘中心
4. 若候选点 col > 当前椎间盘中心 col → 更新；同步重算相邻椎体中心点

#### fan_scanner.py — Step4 / Step4c

##### Step4：终板扫描（双模式）

当前默认使用 **disc 模式**（`ep_scan_mode='disc'`）：终板用矩阵扫描、前缘用扇形扫描。`fan` 模式仍保留以供回退。

**disc 模式（ep_scan_mode='disc'）**：

以椎体中心为原点，取上下两个 junction_pt 沿皮质线2-2构造起点连线（椎体中心 → 皮质线2-2路径中点），扫描方向垂直于该连线：

| 方向 | 参数 | 扫描函数 |
|------|------|---------|
| 上终板（-scan_dir） | `scan_up_mm=30mm`，`drop_ratio=0.35` | `scan_disc_endplates`（矩阵扫描） |
| 下终板（+scan_dir） | `scan_dn_mm=30mm`，`drop_ratio=0.35` | `scan_disc_endplates`（矩阵扫描） |
| 前缘（扇形扫描） | `fan_half_deg=50°`，`scan_ant_mm=40mm`，`ant_drop_ratio=0.35`（S1 为 0.20） | `_scan_normal_descent_ant` 或 `_ant_diag` 版 |

前缘扇形扫描的中心射线角度由 `_calc_ant_angle_deg` 计算（基于椎间盘连线法线或皮质线2-2局部法线）。`scan_results[vi]['fan_params'] = None`（disc 模式无扇形参数），前缘扇形参数独立存储于 `ant_fan_params`。

**fan 模式（ep_scan_mode='fan'）**：

以椎体中心为原点，三个扇形方向扫描：

| 方向 | 参数 | 默认函数 | 最后椎体 < 65° 时 |
|------|------|---------|----------------|
| 上方（上终板）| `fan_half_ep_deg=60°`，`scan_up_mm=30mm` | `_scan_normal_descent` | `_scan_normal_descent_diag` |
| 下方（下终板）| `fan_half_ep_deg=60°`，`scan_dn_mm=30mm` | `_scan_normal_descent` | `_scan_normal_descent_diag` |
| 腹侧（前缘）  | `fan_half_deg=50°`，`scan_ant_mm=40mm` | `_scan_normal_descent_ant` | `_scan_normal_descent_ant_diag`（单像素确认，drop=0.20） |

> **drop_ratio 场景分离**：上/下终板扫描使用 `drop_ratio=0.25`，前缘扫描使用独立参数 `ant_drop_ratio=0.35`（S1 最后椎体降为 0.20）。

**射线角度计算（_calc_ant_angle_deg）**：
- 有椎间盘 top/bot：以椎间盘连线为基准，法线朝腹侧（n_col < 0）
- 无椎间盘：投影到皮质线2，取局部法线方向

**扇形方向语义（坐标系）**：
```
射线角度 = atan2(dr, dc)，dr=sin(angle), dc=cos(angle)
angle 增大 → dr 更负（row分量更负）→ 朝头颅方向（superior）
angle 减小 → dr 更正（row分量更正）→ 朝尾骨方向（inferior）
```

**fan_params 结构**（存储于 `scan_results[vi]['fan_params']`）：
```python
{
  'center': (cr, cc),
  'up':  {'angle': up_angle_deg,   'half': fan_half_ep_deg, 'scan_mm': 30.0},  # 60°
  'dn':  {'angle': down_angle_deg, 'half': fan_half_ep_deg, 'scan_mm': 30.0},  # 60°
  'ant': {'angle': ant_angle_deg,  'half': 50.0,            'scan_mm': 40.0},
}
```

**局部信号采样（_sample_ant_local_signal）**：  
对最后2个椎体（`vi >= n_total - 2`），扫描前缘扇形中间3条射线，统计局部 `ant_low_mean` 和 `ant_high_mean`，用于替代全局参考值，提升骶椎区域适应性。

**前缘扫描函数选择 + S1 专属参数（ant_diag_confirm）**：  
对最后一个椎体（`vi == n_total - 1`），若 Step3.5 计算的 `angle_last < 65°`，则：
- 扫描函数切换为 `_scan_normal_descent_ant_diag`（**单像素信号确认**，而非标准对角均值确认）
- 下降沿阈值 `drop_ratio` 从 0.35 降至 **0.20**（S1 前缘信号对比度低，需降低触发门槛）
否则使用标准 `_scan_normal_descent_ant`（纵向两两组合确认），drop_ratio=0.35。

**终板扫描函数选择（ep_diag_confirm）**：  
与 `ant_diag_confirm` 同条件绑定。`angle_last < 65°` 时，上/下终板扫描同步切换为 `_scan_normal_descent_diag`（`/` 对角信号确认），否则使用标准 `_scan_normal_descent`（横向均值确认）。

日志打印：
```
[Step4前缘] 最后椎体后缘角=xx.x°<65° → 启用对角信号确认（前缘+终板）
```

##### Step4c：_verify_ant_pts_forward（前缘候选点二次校验）

**触发条件**：末尾两椎体汇合点连线夹角 **< 65°**（各自独立判断，≥ 65° 高倾角骶椎区域跳过）

**角度范围过滤**：
- 过滤区间：`delta ∈ [+25°, +50°]`（delta = ray_angle - ant_angle_deg，折叠到 ±180°）
- 过滤区间内的点 **直接删除**（不做信号判断）
- 其余点直接保留
- 角度来源兼容两种模式：fan 模式读 `fan_params['ant']['angle']`，disc 模式回退读 `ant_fan_params['angle']`

**日志输出**：
```

[Step4c] 最后椎体前缘二次校验: X → Y 点保留
```

#### anterior_edge.py — Step5

##### cluster_correction_pts（终板聚类）

去倾斜密集区间聚类 + col排序MAD过滤+移动均值平滑：
1. 估算点集倾斜斜率 k（按offset排序后差分中值）
2. 去倾斜变换：`row_flat = row - k × offset`
3. **5mm** 滑动窗口找最密区间（`win_row_mm=5.0`）
4. 按 **col 排序**（终板线以 col 为自变量，近似水平骨皮质）
5. MAD 过滤 row 离群点（`|row - median_row| < 3×MAD`）
6. 移动均值平滑 row（`smooth_mm=5mm`），col 轴保持原值不平滑
7. **弧长超限回退**：若聚类后总弧长 > 13.0mm，按 40% 弧长窗口重聚类（`fallback_win = total_arc × 0.4`）

##### 最后椎体下终板法线构造

`is_last` 且 `last_ant_angle_deg ≤ 50°` 时，跳过聚类，调用 `_make_inf_from_normal` 用法线直接构造：
- 从 `junction_pts[-1]` 投影到皮质线2-2
- 沿法线方向：背侧 5mm + 腹侧 35mm
- 否则 → 正常聚类

##### cluster_anterior_edge（前缘聚类）

以上终板汇合点和下终板汇合点连线为法线基准轴：
1. 将所有前缘候选点投影到法线轴
2. 仅保留投影值 ≥ 15mm（`offset_min_mm`）的点（排除近端伪影）
3. 滑动 `win_w_mm` 窗口找最密区间，窗口宽度按后缘夹角动态映射：

   | 椎体位置 | 夹角 40° | 夹角 80° | 公式 | angle=None |
   |----------|---------|---------|------|-----------|
   | 最后椎体 | 20mm | 5mm | `20 - (clamp(angle,40,80)-40)/40×15` | 12mm |
   | 倒数第二 | 20mm | 5mm | 同上 | 8mm |
   | 其余椎体 | — | — | 固定 | 5mm |

4. 按 row 排序后移动均值平滑（`smooth_mm=3mm`）

##### _smooth_ant_line（前缘全段平滑）

由 `chain/vertebra_chain.py` 调用（通过 `_stitch_ant_line`）：
1. MAD 过滤（col维度，3σ 剔除异常点）
2. 线性插值填满所有整数行
3. 三步移动均值：15mm → 10mm → 5mm

### 5.4 chain – 椎体链路

#### vertebra_chain.py — Step6：build_vertebra_chain

**A. 前缘线拼接 + 两端法线延伸**（`_stitch_ant_line` + `build_vertebra_chain` 内联逻辑）：
- 各椎体聚类前缘段 → MAD平滑 → 插值 → 两端法线延伸 → 建LUT
- **两端法线延伸逻辑**（需 `c2_rows/c2_cols/junction_pts/disc_centers` 全部有效时启用，否则回退底端斜率延伸20mm）：
  1. 确定前缘线端点到皮质线2-2的**法线偏移量** `d_offset`（欧氏最近点投影）
  2. 沿皮质线2-2走向步进，每步保持 `d_offset` 横向偏移，生成延伸段前缘点
  3. **顶端终止**：皮质线2-2弧长坐标到达 `disc_centers[0]`+`junction_pts[0]` 投影均值弧长 **−5mm** 处停止
  4. **底端终止**：皮质线2-2弧长坐标到达 `disc_centers[-1]`+`junction_pts[-1]` 投影均值弧长 **+5mm** 处停止
  5. 延伸段标记为 `'extrapolated'`，坐标超出图像边界时丢弃

**B. 终板线延长**（_extend_ep_line）：直接使用 `cluster_correction_pts` 输出的聚类平滑点（已在 Step5 内完成 MAD过滤+移动均值平滑，此处不再重复平滑），两端延伸25mm，夹角约束 ±20°（防高度倾斜），超界截断

**D. 四交点求解**（_find_crossing）：
- `sup_ant` = 上终板延伸线 ∩ 前缘线
- `sup_post` = 上终板延伸线 ∩ 皮质线2-2
- `inf_ant` = 下终板延伸线 ∩ 前缘线
- `inf_post` = 下终板延伸线 ∩ 皮质线2-2

交点采用**符号变化法**（_find_crossing + _build_lut_from_line）：
- 将参考线（前缘线/皮质线2-2）构建为 `row→col` 查找表（LUT）
- 对终板延伸线各点计算 `diff = ep_col - lut[row]`
- 检测相邻点 diff 符号反转即为穿越，线性插值得交点坐标
- 相比线段求交法，即使终板线已越过参考线也能正确检测

**D. 角度计算**：
- `ant_angle_deg`：前缘连线（sup_ant→inf_ant）与水平线夹角（绝对值）
- `post_angle_deg`：后缘连线（sup_post→inf_post）与水平线夹角
- `sup_angle_deg`：上终板连线（sup_ant→sup_post）与水平线夹角（`atan2(|dr|, dc)`）

**E. 椎体命名规则**：

```
使用最后两个完整椎体的 sup_angle_deg 判断是否为 S 椎体：

  sup_angle ≥ 30°        → S椎体（直接判定）
  sup_angle ∈ [10°, 30°) → 灰色区间：以下两个条件满足任一即为 S 椎体：
                            a) 宽度比 = sup_width / inf_width ≥ 1.2
                            b) 高宽比 = height / avg_width   ≥ 1.2
                               (height = 上下终板中点直线距离)
                               (avg_width = (sup_w + inf_w) / 2)
  sup_angle < 10°         → L椎体（直接判定）

从下往上命名（四分支）：

  分支1：last=S，sec=S     → S2, S1, L5, L4...
  分支2：last=S，sec=L     → S1, L5, L4...
  分支3：last=L，sec=S     → ⚠️ 解剖矛盾，仍按 S2,S1,L5 命名（sec 判为假阳性忽略）
  分支4：last=L，sec=L     → L5, L4, L3...

灰色区间内双条件为 OR 关系：width_ratio ≥ 1.2 或 hw_ratio ≥ 1.2 → S，均 < 1.2 → L5。
```

## 6. 核心算法步骤详解

### Step2 主扫描 offset 分段线性渐变

```
行程比例 t = (当前行 - 起始行) / 总行程  ∈ [0, 1]

if t ≤ 0.6:
    offset = 1.0 + (t/0.6) × (3.0 - 1.0)   # 1mm → 3mm 线性渐变
else:
    offset = 3.0                              # 固定 3mm
```

设计意图：脊柱上段（胸腰段）汇合点靠近皮质线，offset 从 1mm 起步；  
腰骶段椎间盘空间增大，offset 增加到 3mm，更准确定位低信号带。

### Step2b 末尾实扫19档offset策略

```python
for off_mm in range(-2, 17):   # -2mm ~ 16mm，步进1mm，共19档
    off_px = off_mm / pixel_spacing
    ar = c2r[i] + n_row * off_px   # 沿法线偏移
    ...
    if 信号 < thresh:
        break   # 找到第一个低信号档位即停止
```

- 负值（-2mm~0mm）：向背侧探测，适应骶椎区域皮质线与椎间盘间距极小的情况
- 正值（1mm~16mm）：向腹侧逐档搜索，范围较主扫描更宽

### Step4c 前缘二次校验角度范围过滤

```python
# 每个候选点的射线角度 delta（相对于扇形中心）
delta = ray_angle - ant_angle_deg
# 折叠到 (-180°, 180°]
...
# 过滤区间判断
in_filter_zone = (25° - ε ≤ delta ≤ 50° + ε)
```

**设计意图**：头颅侧上半扇形的上25°区间（+25°~+50°），对应椎体前上角区域。该区域最容易因腹侧软组织（脂肪/肠道）产生假阳性前缘候选点，直接删除以降低聚类误判。中心射线和尾骨侧射线不做过滤。

### Step6 四交点符号变化法求交

终板延伸线与前缘线/皮质线2-2的交点，采用符号变化法（替代旧版线段枚举求交）：

```python
# 1. 将参考线构建为 row→col 查找表（LUT）
ref_lut = {int(round(r)): c for (r, c) in ref_pts}

# 2. 遍历终板延伸线按 row 排序的各点
for (r, c) in ep_sorted:
    ri = int(round(r))
    ref_c = ref_lut.get(ri)
    diff = c - ref_c          # 正：ep在参考线右侧；负：ep在参考线左侧
    if prev_diff * diff <= 0:  # 符号反转 → 穿越
        frac = |prev_diff| / (|prev_diff| + |diff| + ε)
        交点 = 线性插值(prev_pt, cur_pt, frac)
```

**设计意图**：旧版线段求交需要行范围重叠才能相交，当终板延伸线已越过参考线（骶椎区域上终板倾角大时常见）会漏检。符号变化法只要检测到 diff 符号反转即可，对越界情况鲁棒。

## 7. 坐标系与方向约定

| 轴 | 方向 | 说明 |
|----|------|------|
| row | 向下为正 | 图像行索引，0 = 图像顶部 |
| col | 向右为正 | 图像列索引，0 = 图像左侧 |
| 腹侧 | col 减小（向左） | 矢状位图像中椎体前方 |
| 背侧 | col 增大（向右） | 矢状位图像中椎管侧 |
| 头颅侧 | row 减小（向上） | superior 方向 |
| 尾骨侧 | row 增大（向下） | inferior 方向 |

**扇形角度语义**：

```
angle = atan2(dr, dc)
dr = sin(angle)，dc = cos(angle)

angle 增大（+方向） → dr 变负 → 射线朝头颅侧（superior）
angle 减小（-方向） → dr 变正 → 射线朝尾骨侧（inferior）
```

**法线方向**（`_project_to_c2` 返回）：
- 切线 `(t_dr, t_dc)` 顺时针旋转90° → 法线 `(n_row, n_col) = (t_dc, -t_dr)`
- 强制 `n_col < 0`（朝腹侧）

## 附录 A：版本变更记录

### V15.5（2026-03）

**1. Step2 信号确认：三点均值 → 两两组合任一满足**

四个 `_scan_normal_descent*` 函数的 Step2 确认逻辑从「三点均值」改为「两两组合任一满足」：
- `_scan_normal_descent`：`(c-1,c)` 或 `(c,c+1)` 任一均值 < 阈值
- `_scan_normal_descent_diag`：`(r,c)+(r-1,c+1)` 或 `(r,c)+(r+1,c-1)` 任一均值 < 阈值
- `_scan_normal_descent_ant`：`(r-1,r)` 或 `(r,r+1)` 任一均值 < 阈值

旧版三点均值会平均掉非对称信号下降，新版两两组合任一满足可捕获单侧信号突变。

**2. S1 前缘信号确认：对角配对 → 单像素直判**

`_scan_normal_descent_ant_diag` 的 Step2 从 `\` 对角两两组合均值改为**当前像素单点信号**直判。S1 骶椎前缘骨骼-软组织过渡区信号对比度低，单一像素更灵敏。

**3. S1 前缘下降沿阈值动态适配**

最后椎体满足角度校验（`angle_last < 65°`）时，前缘扫描 `drop_ratio` 从 0.35 降至 **0.20**，适应 S1 区域骨骼→软组织信号降幅较小（仅 20-30%）的特点。

**4. 切片优选 Step3 四重形态校验**

桥接候选区域校验从单一宽度比扩展为四重：
- ① 宽度比（0.5~1.5）
- ② 左边界自洽（相邻行跳变 ≤ max(3px, 0.20×med_w)，不依赖 green 尾部比对）
- ③ 宽度一致性 CV ≤ 0.50
- ④ 细长比 ≥ 1.5

其中 ② 左边界自洽彻底重构：旧版取 green 尾部左边界趋势比对候选头部左边界——但桥接点双方均为截断面，趋势本身不真实。新版仅校验候选区域自身左边界相邻行渐变连续性。

**5. S1 局部信号日志输出**

最后两个椎体的局部 `low_mean/high_mean` 采样值现在打印到日志，格式：
```
[局部信号] 椎体6(2nd-last) ant_ang=158.5°  local_low=105.0  local_high=214.0  (全局 low=61.9 high=205.0)
[局部信号] 椎体7(last) ant_ang=123.4°  local_low=125.1  local_high=276.5  (全局 low=61.9 high=205.0)
```
